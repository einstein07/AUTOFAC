\doxysubsection{Detailed Description}
Class for describing a heuristic for positioning an agent before resource pick-\/up. 

\doxylink{classrobogen_1_1PickUpPositioningHeuristic}{Pick\+Up\+Positioning\+Heuristic} inherits from \doxylink{classrobogen_1_1Heuristic}{Heuristic} to describe a heuristic for positioning a robot before picking up resources once agents are within a certain distance to said resources located outside the gathering zone.

\doxysubsection{Detailed Description}
Class for viewing a simulation. 

\doxylink{classrobogen_1_1Viewer}{Viewer} wraps the Open\+Scene\+Graph code, and should be used by other classes that wish to visualize a simulation.

Previously we had separate executables for Server, Server\+Viewer, and File\+Viewer. This involved a lot of duplicate code, so this class aims to be a plugin to enable visualization or not.

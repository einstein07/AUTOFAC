Struct to describe Neural Network representation. 

Given m input neurons and n output neurons m \texorpdfstring{$<$}{<}= MAX\+\_\+\+INPUT\+\_\+\+NEURONS n \texorpdfstring{$<$}{<}= MAX\+\_\+\+OUTPUT\+\_\+\+NEURONS

\doxysubsection{Detailed Description}
Class to describe a \doxylink{classrobogen_1_1RacingScenario}{Racing\+Scenario}. 

The robot that can cover the longer distance in the given simulation time wins. The distance is computed as the euclidean distance from the starting to the ending position computed using as reference the core component.

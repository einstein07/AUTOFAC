Detailed Description

Class for describing a heuristic for positioning an agent before resource pick-up.

PickUpPositioningHeuristic inherits from Heuristic to describe a heuristic for positioning a robot before picking up resources once agents are within a certain distance to said resources located outside the gathering zone.
